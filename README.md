How to run the application
===========================

1. Start the emulator:
1.1. Go to C:\Android\emulator>
1.2. emulator -avd Pixel_API_28_AOSP 

2. To start Metro, run npx react-native start.

3. npx react-native run-android

r - Reload
d - Debug
